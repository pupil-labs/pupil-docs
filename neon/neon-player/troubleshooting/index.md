# Troubleshooting

Did you encounter an issue while using Neon Player?

Let us know on Discord/ Email or by [creating an issue on our GitHub repository](https://github.com/pupil-labs/neon-player-beta/issues).

Please include a detailed description of the issue, steps to reproduce it, and attach the latest log file from Neon Player (see [Settings](../settings/) for more details on where to find this). This will help us investigate and resolve the issue as quickly as possible.
